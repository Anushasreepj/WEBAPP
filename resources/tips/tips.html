• Reset often in the beginning. A lot of different behaviors will emerge from the same settings in the first minute or so.

  • Increase the window size, which increases the board size. A bigger game board usually facilitates evolution.

  • Zoom in/out and use "clip to view" button to resize the grid.

  • You can zoom and clip to isolate interesting organisms.

  • Flip through the color schemes in the display panel.

  • If you see something interesting you can save your progress using the File->Save Genome menu. This saves 20 randomly chosen organisms, capturing any battles.

  • Load 2 or 3 different files using File->Load Additional Genome to create a competition between saved organisms. Competitions use the same Rules settings for all organisms, and rules are updated every time you load.

  • You can browse, load, and share user generated genomes by downloading the SproutLife Genome Gallery from github.com/ShprAlex/ SproutLife-Gallery

  • To create fun evolutionary cascades: 1. Decrease the competitive target age by several points. 2. Increase the mutation rate.

  • Try the other seed types. Bentline to U3x3 and Bentline to Plus take a longer time to get going, but they are more prolific at smaller competitive target ages. Boxlid and Boxhat are more likely to generate echosystems.

  • The "Competitive2 Wild" mode produces the most interesting patterns.

  • Keep the speed below maximum for smoother animation. The maximum speed saves time by not painting as often, and has no other effect.

  • Check out the File->Export GIF option!

  • See what's happening in the Cell Layer and Genome Layer. Slow down the speed to appreciate what's going on. Turn on Visual Sprout Mode in Rules and Outline Seeds & Young to see seeds turn into sprouts.

  • Keep track of Competitive Score in statistics. You'd be surprised how high this can get.

  • More historical data gets logged to the console that you can see if you run SproutLife.jar from the command-line.

  • See the README at SproutLife.org for more details.
